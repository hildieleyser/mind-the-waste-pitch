# Mind the Waste

A fridge sensor that eliminates food waste with AI. This repo holds the pitch deck and a clickable prototype of the app.

## What's here

| File | What it is |
| --- | --- |
| `index.html` | Landing page with links to both |
| `deck.html` | The pitch deck (scroll or use arrow keys) |
| `prototype.html` | Interactive app walkthrough (fully clickable) |

Everything is self-contained HTML with images embedded, so it works offline with no build step or install.

## How to open it

**Easiest:** download this repo (green **Code** button → **Download ZIP**), unzip it, and double-click `index.html`. It opens in your browser.

**Or clone it:**

```bash
git clone <this-repo-url>
cd mind-the-waste-pitch
open index.html      # macOS
# or just double-click index.html in your file browser
```

From the landing page, pick **Pitch deck** or **Interactive prototype**.

### Prototype tips
- Tap **Connect magnet → Demo Mode** to see live sensor readings.
- **Scan shelf** adds detected food.
- Tap any fridge item to mark it used / frozen / shared / binned.
- **Meal plan** builds a recipe from what's about to spoil.

## Notes
- Best viewed full-screen in a modern browser (Chrome, Safari, Edge).
- The deck is a large single file (embedded images), so give it a second to load first time.
